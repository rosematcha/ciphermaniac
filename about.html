About

Howdy! I'm Reese, the developer of Ciphermaniac. Ciphermaniac is a Pokémon TCG analysis tool focused on per-card meta analysis. I made Ciphermaniac out of a desire to track specific cards, a feature which I didn't see on the other popular tools.

The work on this site is largely built on the incredible work of other members of the community, including:

  • LimitlessTCG - for being the source of the tournament data that powers our analysis
  • TrainerHill - for inspiration on the user interface design and layout
  • TCGCSV - for providing pricing data integration

Without these amazing resources, this project would not exist.

If you have additional features you'd like to see on this site, please shoot me a message on the feedback form and I'll try and get back to you soon as I can! Please note that I am working on this in my spare time, and software development is not my forté, so be nice.
